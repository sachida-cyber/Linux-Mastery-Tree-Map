# GitHub Pages Deployment Ready Site

This folder is prepared for GitHub Pages. Place the contents of this `web_deploy_site` folder into your repository's root (or `gh-pages` branch) and enable GitHub Pages.

Structure:
- index.html
- assets/
  - css/
  - js/
  - img/
  - other/

To deploy:
1. Create a new GitHub repository.
2. Copy these files to the repository root.
3. Commit & push.
4. In repository Settings > Pages, set source to main branch / root.
